VIM 三种命令模式

VIM file 后进入命令模式

  • i 插入，在光标之前插入

  • a 插入，在光标之后插入

  • o 新起一行，在下一行插入

  • O 新起一行，在上一行插入

  • A 在行末尾插入

  • s 删除当前字符，等待执行插入

  • S 删除当前行，等待执行插入

 

其他命令

  • 移动：h 左，j 下，k 上，l 右

  • 跳转到指定行：

    • 88G（命令模式下）

    • :88 （末行模式下）

  • 跳转到文件首行：

    • gg （命令模式下）

  • 跳转到文件尾

    • G（命令模式）

  • 自动格式化程序

    • gg=G（命令模式）

  • 大括号对应

    • % （命令模式）

  • 光标移动至行首

    • 0（命令模式）执行结束，工作模式不变

  • 光标移至行尾

    • $ （命令模式），执行结束，工作模式不变

  • 删除单个字符：

    • x （命令模式）执行结束，工作模式不变

  • 替换单个字符

    • r（命令模式）+ 替换后的字符，执行结束，工作模式不变

  • 删除一个单词

    • dw（命令模式）执行结束，工作模式不变

  • 删除光标至行尾

    • D 或者 d$

  • 删除指定区域：

    • 按V（命令模式）切换为 ”可视模式“，然后用方向选中待删除的区域，按 d 删除区域

  • 删除指定一行

    • 在光标所在行，按 dd （命令）

  • 删除指定 N 行

    • 在光标所在首行，按下 ndd（命令模式）

  • 剪切/复制一行：

    • yy

  • 粘贴

    • p（向后）或者P（向前）

  • 查找

    • 命令模式下：按 / 输入想要搜索的关键字，回车；使用 n 检索下一个

    • 命令模式下：将光标置于任意一个字符上，按 "*" 或者 "#" 进行检索

  • 替换

    • 单行替换：将光标置于待替换行上，进入末行模式，输入 ":s /原数据/新数据"

    • 通篇替换：进入末行模式，输入 ":%s /原数据/新数据/g"

    • 指定行替换：进入末行模式，输入 ":29,35s /原数据/新数据/g"

  • 撤销、反撤销

    • 命令模式 u、ctrl + r

  • 分屏

    • 末行模式 :sp 横屏分 :vsp 竖屏分

    • Ctrl + w + w 在多个窗口间切换

  • 保存全部

    • :qwall 保存全部

    • :qall 不保存，退出全部
